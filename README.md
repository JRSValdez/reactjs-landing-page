## Installation

The project was created using **create-react-app** command.

#### Clone the repo
    git clone https://github.com/JRSValdez/reactjs-landing-page.git

### Change to the root folder

    cd ./reactjs-landing-page

### Run install command

    npm install

### Run the app

    npm start
